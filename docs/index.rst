Bienvenue à la documentation de GridGenerator!
==============================================

Grid Generator est un outil permettant de créer des systèmes de communication à grille de pictogrammes optimaux, écrit par Carlos Vargas.  Il s'agit d'un framework qui permet de lire, modifier et évaluer des grilles de pictogrammes et met en œuvre un algorithme de Machine Learning (algorithme génétique) pour trouver les grilles les plus performantes. La métrique utilisée pour évaluer les grilles est le coût de production d'une phrase donnée en entrée.
